## O que é o projeto
Esta atividade consiste no desenvolvimento de um software em linguagem C que simula as operações básicas de um Caixa Eletrônico (ATM). O objetivo principal é aplicar os conceitos fundamentais de programação estruturada em um cenário do mundo real. O projeto é realizado em equipe.

## Como funciona o projeto
O sistema opera em um ciclo contínuo de interação com o usuário através do console (terminal), funcionando da seguinte maneira:

Menu Interativo: Ao iniciar, o programa apresenta um menu visual com as opções disponíveis (Consultar Saldo, Sacar, Depositar e Sair).

Processamento de Escolhas (Estrutura de Decisão): O usuário digita o número da operação desejada. O sistema valida se a opção é correta e direciona para a funcionalidade correspondente.

## Operações Financeiras:

Consulta: Exibe o saldo atual em conta.

Saque: Verifica se há saldo suficiente antes de debitar o valor solicitado. Se não houver, o sistema bloqueia a operação e avisa o usuário.

Depósito: Adiciona o valor informado ao saldo atual.

Fluxo de Repetição: Após finalizar qualquer operação, o sistema limpa a tela (ou o buffer) e exibe o menu novamente, permitindo que o usuário faça novas movimentações até que decida voluntariamente encerrar o programa digitando a opção "Sair".
